About project

This is a service to create, share, view and like pictures generated based on mathematical sets. This project demonstrates how mathematical functions can generate complex and visually stunning patterns.

As an example, the Mandelbrot set is created by iterating the function f(z) = zÂ² + c for complex numbers. Each point on the canvas is colored based on how quickly the iteration diverges (if at all), creating the intricate patterns characteristic of fractals.

The same for other fractals with the difference in used function.
